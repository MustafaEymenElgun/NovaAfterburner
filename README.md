# NovaAfterburner
NOVA Afterburner v2.0 - Pro System Monitor
NOVA Afterburner, Windows sistemleri için geliştirilmiş, düşük kaynak tüketimli ve yüksek görsellik sunan bir sistem izleme aracıdır. Standart görev yöneticisinden sıkılanlar için MSI Afterburner estetiğiyle modern Python kütüphanelerini birleştirir.

🔥 Temel Özellikler
Gerçek Zamanlı CPU & RAM İzleme: İşlemci ve bellek yükünü canlı grafiklerle takip edin.

Ağ Trafiği Analizi: Anlık indirme (download) hızını KB/s cinsinden görüntüleyin.

En Çok Kaynak Tüketen Uygulamalar: Bilgisayarınızı o an en çok yoran ilk 5 işlemi anlık olarak listeleyin.

Modern Dashboard: CustomTkinter ile hazırlanmış, göze hitap eden Dark Mode arayüz.

Asenkron Çalışma: Threading mimarisi sayesinde arayüzde donma yaşanmadan veri takibi yapılır.

Dinamik Ölçeklendirme: Grafikler ve değerler sistem yüküne göre anlık olarak güncellenir.


🛠️ Kurulum
Projeyi yerel makinenizde çalıştırmak için aşağıdaki adımları izleyin:

Depoyu klonlayın:

Bash
git clone https://github.com/kullaniciadi/nova-afterburner.git
cd nova-afterburner
Gerekli kütüphaneleri yükleyin:

Bash
pip install customtkinter psutil matplotlib
Uygulamayı başlatın:

Bash
python main.py
📊 Kullanılan Teknolojiler
Python: Ana programlama dili.

CustomTkinter: Modern ve özelleştirilebilir GUI bileşenleri.

Psutil: Sistem bilgilerine (CPU, RAM, Network) erişim.

Matplotlib: Verilerin profesyonel grafiklere dökülmesi.
